## 1. 交易策略

### 1.1 核心开仓信号

所有策略均采用 **30 分钟级别三买信号** 作为开多条件：

```
30分钟_D1_三买辅助V230228_三买_任意_任意_0
```

过滤条件：排除涨停标的

```
30分钟_D1_涨跌停V230331_涨停_任意_任意_0
```

### 1.2 退出规则对比（案例 19c）

| 规则 | 说明 |
|------|------|
| baseline | 笔向下平仓（`30分钟_D1_表里关系V230101_向下`） |
| fixed_10 | 固定持仓 timeout=10 根 30 分钟 K 线后强制平仓 |
| fixed_20 | 固定持仓 timeout=20 根 30 分钟 K 线后强制平仓 |
| stop_300 | 同 baseline，止损 300 BP |
| stop_150 | 同 baseline，止损 150 BP |

共同参数：
- 开仓间隔 `interval = 4 小时 = 8 根 30 分钟 K 线`
- `t0 = False`（不允许 T+0，跨日才能平仓）
- 止损 `stop_loss = 300 BP`
- 手续费 `fee_rate = 0.0002`
- 权重类型 `ts`，年交易日 `252`

### 1.3 未来函数诊断（脚本 compare_execution_lag）

对比同一 fixed20 策略的三种执行方式：

1. **同 bar close**：当前 Rust 引擎默认行为
2. **延后 1 根 open**：用下一根 K 线开盘价作为成交价
3. **延后 1 根 close**：用下一根 K 线收盘价作为成交价

### 1.4 板块扫描（案例 19g）

基于 `19f_sector_scanner` 产出的六个板块 CSV 与 `19e` 的个股/ETF 组合结果，汇总 2025-2026H1 期间：

- 板块平均表现
- 各板块 combined 策略夏普最优品种
- 重点关注标的（510300.SH、601012.SH、688008.SH）对比

## 5. 结论

1. **交易次数整数化修复生效**：`19g_sector_report.py` 与 `report/generate_backtest_report.py` 产出的 `trades` 列均为整数，PDF 报告已重新生成。

2. **timeout 差一修改已回退**：当前 `position.rs` 恢复为 `bar_id - ev_bar_id > self.timeout` 的上游行为。

3. **fixed_20 规则表现突出**：
   - 2020-2024 段年化 6.02%，夏普 0.57
   - 2024-2025 段年化 10.41%，夏普 0.91
   - 明显跑赢 baseline 和 tighter 止损规则

4. **未来函数风险在 fixed20 上不明显**：延后 open/close 成交并未显著削弱收益，延后 close 反而表现更优，说明策略对执行滑点具有一定鲁棒性。

5. **板块层面**：
   - 半导体、科技板块 2025-2026H1 表现最佳
   - 688008.SH（澜起科技）是重点关注标的中收益最高的，30_60_fixed20 策略年化 63.79%，夏普 1.73
   - 医药、消费板块在样本期内为负收益

6. **持仓 K 线数观察**：fixed_10 / fixed_20 的实际持仓 K 线数平均在 34 / 52 左右，远大于 timeout 设定值，主要原因是 `t0=False` 下 timeout 触发当日无法平仓，需顺延至下一交易日。若希望严格按 timeout 根 K 线平仓，需要考虑是否允许 T+0 或调整 timeout 计算逻辑。
